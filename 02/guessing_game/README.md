let is used for creating variables.

in rust by default variables are by default immutable;

use mut to make a variable mutable;

```
let x = 5; //=> immutable 
let mut y = 6; //-> mutable
```
for example bellow code is wrong.
```
let x = 10;
x = 11;
```
